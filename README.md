# React Native Improving

## Requirements
* [OpenJDK](https://openjdk.java.net/)
* [Node](https://nodejs.org/es/)
* [npm](https://www.npmjs.com/)
* [Android Studio](https://developer.android.com/studio)
* [xCode](https://developer.apple.com/xcode/)
* [CocoaPods](https://cocoapods.org/)

## installation

### Download the repository
Run the following commands in a terminal:
``` bash
git clone https://github.com/NicoPalazzesi/react-native-improving.git
cd react-native-improving
```

### Install the dependencies
Run in a terminal
``` bash
npm install
npm start
```

### Android
Open your preferred Android simulator or let React Native choose one by default.
You can use emulators like Android Studio, Genymotion, or any other you prefer.


Run the command:
``` bash
npm run android
```

### iOS
Open your preferred iOS simulator or let React Native choose one by default.

Run the commands:
``` bash
cd ios/ && bundle install && bundle exec pod install
cd ../
npm run ios
```
